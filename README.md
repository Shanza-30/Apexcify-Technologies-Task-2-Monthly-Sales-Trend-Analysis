## 📌 Overview

This project is part of Apexcify Technologies – Task 2, where sales data is analyzed and visualized to understand monthly performance trends. Using Python libraries, raw data is transformed into meaningful insights through effective visualization.


## 🎯 Objective
Analyze sales data from a CSV file
Identify monthly sales trends
Create a clear and professional line chart
Generate insights for better business decisions


## 🛠️ Tech Stack
Python 🐍
Pandas 📊
Matplotlib 📈


## 📂 Dataset Information

The dataset contains sales records with the following key columns:

Order Date → used to extract month
Sales → used to calculate total monthly revenue


## ⚙️ Project Workflow
Data Collection – Imported dataset from CSV file
Data Cleaning – Converted date column into datetime format
Feature Engineering – Extracted month from order date
Data Aggregation – Calculated total sales for each month
Visualization – Created a line chart using Matplotlib
Insight Generation – Analyzed trends and performance


## 📈 Visualization Output

The final output is a line chart representing monthly sales trends across the year.

![Monthly Sales Chart](monthly_sales.png)


## 🔍 Key Insights
Sales show clear variation across months
Certain months perform as peak sales periods
Some months indicate low sales activity
Trend analysis helps in strategic planning and forecasting


## 🚀 Conclusion

This project demonstrates how data visualization can convert raw data into actionable insights. It helps businesses understand performance trends and supports better decision-making.
